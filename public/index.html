Boonaki's Tea API

  • - Name
  • - Alternate Names
  • - Origin
  • - Image
  • - Caffeine Content (1 cup/8 fl. oz.)
  • - Type or types
  • - Quick description
  • - Taste description
  • - Color description
  • - If tea is a blend, data will include the main ingredients

Routes

Get the organized list Get the unorganized list Query for an individual tea
